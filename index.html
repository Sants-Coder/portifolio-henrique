FIAP PROJECT

"A seção "Meus Cartões" foi desenvolvida como parte do projeto de pós-graduação na FIAP, com foco em uma aplicação de gerenciamento financeiro. Nessa tela, implementei a exibição dinâmica de cartões físicos e digitais, com botões interativos para ações como configurar, bloquear e desbloquear cartões. Utilizei Next.js com Tailwind CSS para garantir responsividade, tema escuro e uma estrutura moderna baseada em componentes reutilizáveis. A interface também inclui modais com lógica de confirmação, proporcionando uma experiência de usuário fluida e intuitiva."
