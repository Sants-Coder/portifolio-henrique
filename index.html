PROJECTS

Why Not?

Mais do que um site, uma experiência sensorial. Um projeto autoral criado para transformar motivação em interface e estética em propósito. Cada elemento visual foi pensado para provocar sentimento, reflexão e identidade.

Este projeto representa minha visão de Front-End como narrativa, impacto e expressão.

Invincible — Color Web

Um experimento visual que evoluiu para uma experiência narrativa interativa. A interface reage às escolhas do usuário, alterando cores, atmosfera e identidade visual em tempo real.

Um projeto que explora como UI pode contar histórias e carregar significado.

Dashboard Bancário — FIAP

Projeto desenvolvido na pós-graduação da FIAP com foco em aplicações financeiras. Interface moderna construída com Next.js e Tailwind CSS, utilizando componentes reutilizáveis, tema escuro e modais interativos.

Ênfase em usabilidade, organização visual e experiência do usuário.

Site para Tatuador — Yago Miranda

Site one-page responsivo desenvolvido para um cliente real. Foco em branding pessoal, estética dark minimalista e conversão.

Inclui menu mobile, FAQ, CTAs estratégicos e integrações com WhatsApp e Google Maps.

BULLY — Tribute Project

Projeto inspirado no clássico da Rockstar Games. Uma imersão visual no universo de Bullworth Academy, explorando personagens, facções e a atmosfera que marcou gerações.

Um tributo à narrativa e identidade cultural dos games.

2PAC — Tribute Website

Site dedicado à vida e legado de Tupac Shakur. Um projeto focado em conteúdo, identidade cultural e impacto social, reunindo discografia, história e representatividade.

First Project — Jump Sky

Meu primeiro site desenvolvido durante a graduação em Jogos Digitais. Um marco inicial que representa o começo da minha jornada no Front-End e a evolução que veio a partir dele.

TEACHING

Sants Coder — Aprendendo Front-End com Projetos Reais

Você é dev e quer entender como esses projetos são construídos na prática? No canal Sants Coder, eu ensino passo a passo como criar aplicações visuais, animações e microinterações usando HTML, CSS e JavaScript.

A ideia não é só copiar código, mas entender o raciocínio por trás de cada decisão.

Hex Glitch Effect

Efeito visual animado criado com CSS e JavaScript. Ensino como trabalhar com keyframes, timing e sensação de glitch para criar interfaces mais expressivas.

Matrix Rain Animation

Animação inspirada no efeito Matrix Rain usando canvas e JavaScript. O foco é lógica, performance e controle visual em tempo real.

Pulse Bars Loader

Loader animado utilizando apenas CSS. Um microprojeto ideal para entender animações, delays e ritmo visual.
